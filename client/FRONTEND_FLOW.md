# NyayaSankalan Frontend Flow

1. User logs in
2. JWT stored in memory / secure storage
3. Role detected
4. Redirect to role dashboard

## Police Flow
- Create FIR
- Case auto-created
- Investigation
- Upload evidence
- SHO reviews
- Submit to court (lock UI)

## Court Flow
- Intake case
- Generate acknowledgement
- Judge views
- Court action
